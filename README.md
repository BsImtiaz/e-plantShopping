# e-plantShopping

## Project Overview

e-plantShopping is a React-based web application designed to provide users with an interactive and user-friendly platform for browsing and purchasing plants online.

The application focuses on modern UI design, smooth navigation, and efficient state management to enhance the overall shopping experience.

## Features

* Browse a variety of plants
* Add items to cart
* Responsive and clean UI
* Built using React

## Tech Stack

* React.js
* JavaScript
* HTML5 & CSS3
